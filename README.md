## 🧠 Project Overview

**AI Fitness Buddy** is an intelligent fitness and nutrition assistant powered by **Google Gemini 2.5 Flash**. It allows users to analyze food images to estimate calorie content and generate personalized diet plans based on individual goals.

This project demonstrates:
- ✅ Seamless AI Integration
- ✅ Effective UX Design
- ✅ Smart handling of Complex Logic within a clean and responsive UI

---

## 🚀 Features

- 🥗 **AI-Powered Nutrition Analysis** – Upload a meal image and get calorie & nutrition insights instantly.
- ⚡ **Personalized Diet Planning** – Create dynamic diet plans using AI prompts based on your goals.
- 🧩 **Zero Backend Infrastructure** – Fully client-side; no server cost.
- 💅 **Modern UI/UX** – Built with React + Vite, styled with Glassmorphism.
- 🔒 **Consistent Data Handling** – AI responses parsed strictly as JSON for reliability.
- 🔑 **Demo Key Support** – Recruiters can try instantly with one click, no setup needed.

## ⚙️ Installation & Setup

### 1️⃣ Clone the Repository
```bash
git clone https://github.com/pskalm/ai-fitness-buddy.git
cd ai-fitness-buddy
```

### 2️⃣ Install Dependencies
```bash
npm install
```

### 3️⃣ Add your API Key
Create a `.env` file in the root:
```
VITE_GEMINI_API_KEY=your_gemini_api_key_here
```
Get your free key from [Google AI Studio](https://aistudio.google.com/app/apikey)

### 4️⃣ Run the Application
```bash
npm run dev
```
Open your browser at 👉 `http://localhost:5173`
